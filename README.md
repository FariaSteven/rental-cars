## Getting Started

### Prerequisites

- Java 11 or later
- Maven 3.8.1 or later

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/FariaSteven/rental-cars
    cd rental-cars
    ```

2. Build the project:

    ```bash
    mvn clean install
    ```

3. Run the application:

    ```bash
    java -jar target/rental-cars.jar
    ```

The API will be accessible at `http://localhost:8080`.
